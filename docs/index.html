convert full/half width, Chinese/Japanese number ... others util helper

npm i str-util

Usage

import * as StrUtil from 'str-util';

See more at test / docs

FullHalf

const str = 'THE ｑｕｉｃｋ， ＢＲＯＷＮ\u3000fox.';

StrUtil.toFullWidth(str);
//=> 'ＴＨＥ　ｑｕｉｃｋ，　ＢＲＯＷＮ　ｆｏｘ．'

StrUtil.toHalfWidth(str);
//=> 'THE quick, BROWN fox.'
